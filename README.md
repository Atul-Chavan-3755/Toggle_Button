#🌗 Dark Mode Toggle Button

A simple and interactive Dark Mode / Light Mode toggle button built using HTML, CSS, and JavaScript.
Click the button to switch the screen to Dark Mode, and click again to return to Light Mode.

#🚀 Features

🌙 Toggle between Dark and Light themes 

🎨 Smooth UI theme transition

⚡ Lightweight and beginner-friendly

🧠 Demonstrates DOM manipulation

📱 Responsive design

#🛠️ Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

#📸 How It Works

User clicks the toggle button.

JavaScript detects the click event.

A CSS class (e.g., dark-mode) is added or removed.

The theme changes dynamically.

#💡 Learning Concepts Covered

Event Listeners (addEventListener)

DOM Manipulation

ClassList Toggle (classList.toggle())

Basic UI Design

CSS Theme Switching

Click the toggle button
